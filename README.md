BXorigcapt Package
==================

LaTeX: To retain the original caption text when using Babel
                                                            
### System requirement

  * TeX format: LaTeX.
  * TeX engine: Anything.
  * Dependent packages:
      - babel

### Installation

  - `*.sty` → $TEXMF/tex/latex/BXorigcapt
